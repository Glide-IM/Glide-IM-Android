# GlideIM Android 客户端

Android UI 部分使用 kotlin 实现, IM 封装成了一个 SDK, 使用 Java 实现.

## 基本功能

- 注册登录
- 好友添加
- 群添加 创建群
- 会话列表
- 联系人列表
- 自动重连
- 消息收发
